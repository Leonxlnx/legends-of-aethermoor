# Legends of Aethermoor

3D Open World Third-Person Action RPG built with Three.js + Vite.

## Run Locally
```bash
npm install
npm run dev
```

## Controls
- WASD: Move
- Mouse: Camera
- Shift: Sprint
- Space: Jump
- Left Click: Attack (3-hit combo)
- Right Click: Dodge Roll
- Q: Lock-On
- E: Continue Dialogue
